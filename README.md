# Izzy Eats - Luxury Food Delivery Landing Page

A modern, high-performance landing page designed for a premium food delivery service. This project showcases clean, semantic structure, responsive layouts, and native JavaScript interactions built entirely from scratch.

## 🛠️ Tech Stack & Features
- **Semantic HTML5 & Custom CSS3:** Built with a dark-mode theme utilizing CSS variables for scalable design token management.
- **Dynamic Scroll Spy Navigation:** Pure JavaScript tracking that updates and highlights the active menu link as the user scrolls.
- **Interactive Search Overlay:** Full-screen modal overlay with focus management and keyboard accessibility (closes instantly with the `Esc` key).
- **Micro-interactions:** Smooth CSS keyframe floating animations and reactive cart badge "pop" states when items are added.
- **Fail-safe Image Fallbacks:** Inline `onerror` handling to gracefully swap missing images with localized emojis, preserving UI integrity.
